# Listen-Eigenschaften

Dieses Verzeichnis enthält Aufgaben, bei denen es darum geht, Listen auf bestimmte
Eigenschaften hin zu untersuchen.

## Herangehensweisen

Die Funktionen in diesem Verzeichnis sind einander sehr ähnlich und verwenden nur leicht
unterschiedliche Herangehensweisen, um eine bestimmte Eigenschaft einer Liste zu ermitteln.
Es geht hier darum, die Lösungen aus den vorherigen Übungen zu wiederholen und anzupassen.

## Für Fortgeschrittene

Fortgeschrittene können sich auch überlegen, wie man die Funktionen auf allgemeinere Weise
gestalten kann, so dass sie für verschiedene Eigenschaften und Listen-Typen wiederverwendet
werden können. Stichworte sind hier z.B. "Lambdas", "Funktionen als Werte" oder "Generics".
